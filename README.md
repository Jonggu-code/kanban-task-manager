# 칸반 태스크 매니저

React + Vite 기반의 칸반 보드 형태 태스크 관리 애플리케이션입니다.
칸반(할 일/진행 중/완료) 보드를 기반으로 태스크를 생성·관리하고, 드래그 앤 드롭으로 상태를 이동할 수 있습니다.
색/필터/정렬로 원하는 태스크를 빠르게 찾을 수 있으며, 데이터와 테마는 localStorage에 저장되어 새로고침 후에도 유지됩니다.

## 트러블슈팅

### 드래그 앤 드롭이 갑자기 동작하지 않는 문제

**현상**

- 카드에 애니메이션(추가/삭제)을 붙인 뒤, 드래그 앤 드롭이 먹지 않거나 카드가 움직이지 않음

**원인**

- `@hello-pangea/dnd`는 드래그 중 카드에 `transform: translate(...)` 인라인 스타일을 적용해 위치를 이동시킴
- 그런데 카드 요소에 `@keyframes` 기반 애니메이션을 직접 적용하면서(특히 `transform`을 사용하는 애니메이션) 라이브러리가 주는 `transform`이 애니메이션에 의해 덮어써져 드래그 이동이 깨짐

**해결**

- 드래그 대상(Draggable root)에는 애니메이션을 적용하지 않고 내부 래퍼 요소에만 애니메이션을 적용하도록 구조를 변경
- 결과적으로 DnD의 `transform`은 유지되고, 카드 진입/퇴장 애니메이션은 정상 동작
